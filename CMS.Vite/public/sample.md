This is a **Blazor Custom Element** that fetches and renders markdown using **Markdig**.

## Features:
- ✅ Easy to use
- ✅ Supports all markdown features
- ✅ Integrates seamlessly with Vite
- ✅ Fetches content from URL specified by a paramter
- ✅ Supports diagnostic styling controlled by a parameter

### HTML Example:
```html
<markdown-viewer markdown-content-path="/sample.md" show-details="true"></markdown-viewer>
```
